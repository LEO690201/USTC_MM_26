# 中国科学技术大学 2026 年春季学期《数学建模》课程作业（beta version）

## 课程简介

本仓库为中国科学技术大学 2026 年春期学期本科生《数学建模》课程作业，旨在培养学生运用数学方法解决实际问题的能力。课程涵盖微分方程建模、优化理论、机器学习算法设计等核心内容。

> 注意：本仓库仅为测试版，实际作业仓库请以课程通知为准！

## 作业概览

| 作业 | 主题 | 难度 |
|------|------|------|
| 1 | 传染病模型 (SIR/SEIR) | ⭐⭐ |
| 2 | 投资组合优化 | ⭐⭐⭐ |
| 3 | 机器学习中的优化算法 | ⭐⭐⭐⭐ |
| 4 | 图论与网络分析 | ⭐⭐⭐⭐ |
| 5 | 时间序列分析与预测 | ⭐⭐⭐⭐ |

## 前置要求

- Python 3.8+
- NumPy, SciPy, Matplotlib
- Pandas, Scikit-learn
- (可选) PyTorch, NetworkX

## 提交要求

1. 代码保存在对应作业文件夹的 `src/` 目录下
2. 报告保存为 `report.md` 或 PDF 格式
3. 命名格式: `学号_姓名_homework{n}.zip`

## 评分标准

- 代码正确性 (40%)
- 数学原理阐述 (30%)
- 结果分析与可视化 (20%)
- 创新性与代码风格 (10%)
